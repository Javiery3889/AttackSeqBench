1. **Initial Access via Spearphishing:**

   - This is a common initial access vector because it exploits human trust and can bypass some technical defenses if the user falls for the phishing attempt.

2. **Mapping the Network:**
   - Once initial access is gained, attackers often perform reconnaissance to understand the target network's structure, identify high-value targets, and find potential paths for lateral movement.
   - Tools and techniques for network mapping include using built-in system commands, scripting, or specialized tools to discover active machines, open ports, shared resources, and more.

3. **Collecting Passwords:**
   - Attackers may seek to collect credentials, such as passwords, to gain access to more systems or to escalate their privileges within the network.
   - Techniques for credential harvesting include capturing plaintext passwords, stealing password hashes, or using tools to extract credentials from memory or stored files.

4. **Obtaining Domain Administrator Privileges:**
   - Domain administrators have high-level privileges in Active Directory environments, allowing them to manage user accounts, install software, and make changes across the entire domain.
   - Gaining domain admin privileges is a significant milestone for attackers as it provides broad access and control over the network.

5. **Using Tools like PsExec and WinExe for Lateral Movement:**
   - PsExec and WinExe are legitimate administrative tools that can be used to execute processes on remote systems.
   - Attackers may abuse these tools to move laterally within the network, spreading their access from the initially compromised system to others.

- Initial access through spearphishing.
- Network mapping and credential harvesting to obtain domain admin privileges.
- Using PsExec and WinExe for lateral movement.

- **Lateral Movement:** With elevated privileges, attackers can use tools like PsExec and WinExe to move laterally and spread their access across the network.
