**Discovery:** This tactic involves techniques used by adversaries to learn about the system and environment they've infiltrated. This could include things like querying system information, enumerating users and systems on the network, and so on. The goal here is to gather intelligence to plan further steps in the attack.

**Defense Evasion:** This tactic covers techniques used by adversaries to avoid detection and analysis. This could involve things like modifying registry settings to hide malicious activities, using encryption or encoding to obfuscate data, or even directly targeting and disabling security software.

**Command and Control (C2):** This is about maintaining communication between the malware and the attacker's command center. This communication is crucial for receiving further instructions, uploading stolen data, downloading additional malware, and more. Essentially, it's the lifeline for the malware once it's on the target system.

**Execution:** This tactic involves techniques used by adversaries to run malicious code on the target system. This could be through exploiting vulnerabilities, using legitimate system tools in malicious ways, or directly executing malicious binaries.
