**Collection:** This tactic involves actions where an adversary gathers information it deems useful for its objectives. This could be data from files, system information, user activity, etc.

**Defense Evasion:** This tactic involves techniques used by adversaries to avoid detection by security mechanisms or to prevent the system from responding to an attack.

**Exfiltration:** This tactic is about the methods adversaries use to steal data from the target network. This could involve moving data to external servers or encoding and compressing data for easier transmission.

**Command and Control:** This tactic involves the communication between the malware and its controller. It's about how the attacker sends commands to the malware and receives data back.
