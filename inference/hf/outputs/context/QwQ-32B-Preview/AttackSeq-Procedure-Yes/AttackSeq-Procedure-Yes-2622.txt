**CTI Outline Analysis:**

1. **Initial Access:**

5. **Credential Access:**

6. **Discovery:**
   - Listed nearby Wi-Fi networks using netsh command.
   - Detected newly inserted devices.
   - Produced file listings for exfiltration.
   - Enumerated running processes in search of browsers.
   - Gathered browser data such as bookmarks and reported window names along with keylogger information.

7. **Collection:**
   - Stole clipboard contents.
   - Copied files from local systems and removable media.
   - Logged keystrokes.
   - Captured screenshots.
   - Staged data in a temporary folder where files and logs were encrypted for collection.

8. **Exfiltration:**
   - Automatically exfiltrated all collected files to remote servers.
   - Data was encrypted with AES before transmission and, in some versions, also base64 encoded.
   - Browser profile data was compressed as .zip files prior to exfiltration.
   - Data was exfiltrated over the same channel used for C&C.
   - Scheduled transfers set up to send data to the C&C server every 10 minutes.
   - Data from all drives could be copied to a removable drive if a special file was present on that drive.
