- **Discovery**: This is about the adversary gathering information about the environment, such as system information, user accounts, and network resources.

- **Command and Control**: This tactic involves communications between the compromised system and the attacker's command and control server. This is how the attacker sends commands to the malware and receives data back.

- **Impact**: This includes actions that result in system disruption, like data deletion, service disruption, or system shutdown.
