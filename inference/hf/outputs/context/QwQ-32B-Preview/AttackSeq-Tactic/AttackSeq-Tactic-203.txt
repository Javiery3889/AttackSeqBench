- Defense Evasion: Techniques used to prevent detection and analysis.

- Credential Access: Techniques used to steal account names and passwords.

- Collection: Techniques used to gather data of interest to the attacker.

- Command and Control: Techniques used to communicate with compromised systems.

- Exfiltration: Techniques used to steal data from the target network.

- Impact: Techniques that result in system disruption.
