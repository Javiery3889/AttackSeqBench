- **T1546.017-Udev Rules**: This technique involves modifying udev rules to execute malicious code when a specific device is connected. This is a more advanced and specific technique that is not typically used in the initial stages of an attack.
- **T1078.001-Default Accounts**: This technique involves using default accounts that come with software or systems. While this can be used to gain initial access, it is not typically associated with targeting specific individuals via fake job offers and resumes.
