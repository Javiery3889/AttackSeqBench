#### 1. Analyze the CTI Report

**Persistence:**
- The group drops further tooling and/or seeks to obtain legitimate credentials to maintain persistent access.
- Anonymizing services are used when using stolen credentials.

**Command and Control:**
- Custom malware (WellMess and WellMail) is deployed.
- WellMess can execute shell commands, upload/download files, and supports HTTP, TLS, and DNS communications.
- WellMail runs commands or scripts and sends results to a C2 server using hard-coded TLS certificates.

**Others:**

- Uses custom malware like WellMess and WellMail.

  - This technique involves using legitimate credentials, which is a step in achieving persistence.
- **D: T1218.009-Regsvcs/Regasm**
  - This technique involves using signed binaries to load malicious code, which is not mentioned in the report.
