3. **Compare and Evaluate:**

   - **T1036-Masquerading:** This technique involves making malicious files or processes appear legitimate to avoid detection. It is often used during the Execution phase to run malicious code without raising suspicion.
   - **T1505.001-SQL Stored Procedures:** This technique involves using SQL stored procedures to execute commands or scripts. It is typically used during the Execution or Persistence phases.

4. **Reasoning:**
