### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Blind Eagle uses a JavaScript downloader that employs ActiveXObject to execute PowerShell commands.
- The group leverages Discord CDNs to host the subsequent stage script.

**Execution:**
- A PowerShell script loads a DLL into memory from an obfuscated and Base64 encoded string.
- The DLL contains a method named Bypass() to bypass AMSI.
- The script checks for the presence of the registry key `HKCU:\software\wow6432node\Microsoft\WindowsUpdate`.
- If the key is absent, it creates the key and sets an obfuscated value.
- It drops two files: `myScript.ps1` and `SystemLogin.bat`.
- A VBScript named `Login1.vbs` is placed in the Startup folder to execute `SystemLogin.bat` upon system start.
- Another batch script deobfuscates and executes `myScript.ps1` using `mshta`.
- A final PowerShell script loads the AMSIBypasser DLL and an AES256 Decryptor to decrypt the registry key content, leading to njRAT (Bladabindi).

**Persistence:**
- A VBScript named `Login1.vbs` is placed in the Startup folder to ensure the execution of `SystemLogin.bat` upon system startup.

**Command and Control:**
- The final PowerShell script leads to njRAT, which establishes command and control over the victim's computer.

**Others:**
- APT Blind Eagle is a suspected South American espionage group targeting Colombian government institutions and corporations.
- They use sophisticated attack techniques, including custom malware, social engineering, spear-phishing, and zero-day exploits.
