**Execution:**

**Discovery:**
- Ran commands to discover system and network information (user accounts, files, folders, privileged groups, remote systems, network configuration).

**Command and Control:**
- Used AntSword webshell for communication and command execution.
- Used AntSword Shell Manager to interact with the webshell.

#### 2. Analyze the Question

The question asks if it is likely that the actors:
- Dumped credentials with a Mimikatz variant.
- Used Impacket's atexec tool.
- Only then used webshells for lateral movement to other systems.

**Sequence of TTPs in the CTI Report:**
1. **Initial Access:** Exploited CVE-2019-0604 to install webshells.

**Evaluation:**
