**Initial Access:**
- GADOLINIUM delivered malicious Access database files via spear-phishing emails.
- The initial file was an Access 2013 database (.accde format) that dropped a fake Word document, an Excel spreadsheet, and a VBA dropper called mm.accdb.core.
- The VBA dropper executed a .NET DLL payload.

**Defense Evasion:**
- GADOLINIUM used cloud services and open-source tools to obfuscate their activity.
- They created a malicious infrastructure using cloud-based technology that can be quickly established and taken down.

**Command and Control:**

- They installed web shells on legitimate websites for C2 or traffic redirection.
- They used PowerShell Empire toolkit to load additional modules via Microsoft Graph API calls, using the attacker's OneDrive account for command execution and result retrieval.

**Others:**
- Contextual information about GADOLINIUM's activities, targeting, and Microsoft's defensive measures.
