### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Exploited a vulnerability in an internet-exposed Solaris server's SSH service to install the SLAPSTICK backdoor.
- Deployed the EVILSUN tool containing a zero-day exploit for CVE-2020-14871 on a Solaris 9 server.

**Execution:**
- Utilized custom virtual machines pre-configured with post-exploitation tools.
- Employed the LEMONSTICK backdoor for command execution and file transfer.

**Persistence:**
- Established persistence using the SLAPSTICK backdoor with a secret hard-coded password.
- Placed a copy of a legitimate pam_unix.so file and SLAPSTICK in the /lib64/security folder.

**Privilege Escalation:**
- Escalated privileges using the SLAPSTICK backdoor and open source tools like Mimikatz.
- Installed three backdoors: SLAPSTICK, TINYSHELL, and OKSOLO.

**Credential Access:**
- Collected credentials using the SLAPSTICK backdoor and Mimikatz.

**Discovery:**
- Maintained various tools for internal reconnaissance, including network scanners, exploits, and reconnaissance tools.
- Employed a SPARC executable for reconnaissance.

**Lateral Movement:**
- Moved laterally using stolen credentials and tools like ProxyChains to download PUPYRAT.
- Used IMPACKET with SMBEXEC in a Windows environment for remote command execution.
- Moved laterally via RDP to a Windows server.

**Command and Control:**
- Used SSH Port Forwarding to maintain access to external infrastructure.
- Deployed a custom QEMU Virtual Machine for direct access to the command and control server.

**Others:**
- No evidence of data exfiltration.
- ROLLCOAST ransomware deployment observed in one case, but not attributed to UNC1945.
