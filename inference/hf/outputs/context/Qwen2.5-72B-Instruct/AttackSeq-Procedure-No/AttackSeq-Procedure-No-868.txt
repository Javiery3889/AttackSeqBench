### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Execution:**
- The Sednit group's operators took control of the backdoor and began sending commands immediately after the victim executed the downloader.
- Commands included taking screenshots, gathering system information, network scanning, and executing various system commands to collect data.
- A Stage-1 downloader was used to download and execute a new downloader, which then downloaded a Delphi backdoor.
- The backdoor's configuration was stored in the resource section, split into four different hex-encoded, encrypted blobs.

**Persistence:**
- COM object hijacking was used for persistence.
- Registry keys were added for a custom backdoor executed using the CMD_EXECUTE command.
- The backdoor was installed for a few hours and then removed.

**Defense Evasion:**
- Malware deleted files (binaries and files created) after usage.
- Processes were killed to disable security tools.

**Credential Access:**
- Operators uploaded and used dumpers on victims' machines to target browsers and versions of Microsoft Outlook from 1997 through 2016.
- Retrieved databases related to credentials from software such as Firefox, The Bat! email client, and Opera.

**Discovery:**
- Gathered information about the victim's computer and environment, including registry keys enumeration, listing running processes, and using system commands to gather information.

**Collection:**
- Scanned for files matching specific extensions on the local system, network shared drives, and removable media.
- Staged data by creating a file containing paths of all files to exfiltrate.
- Features for input capture and screen capture.

**Command and Control:**
- Downloaders and backdoors communicated with the C&C server using commonly used ports 80 or 443.
- Data sent were hex-encoded and encrypted with AES or a custom algorithm.
- A fallback C&C server was embedded in the configuration.

**Exfiltration:**
- Data were automatically prepared with all file paths to retrieve and sent to the C&C server.
- Data were hex-encoded and encrypted with a known algorithm or a custom one.
