### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Threat actors exploited CVE-2019-0604 in SharePoint to install webshells, including AntSword.

**Execution:**
- Used AntSword webshell to run commands for system and user discovery.
- Uploaded tools like cURL, a custom Mimikatz variant, and Impacket's wmiexec and atexec.

**Defense Evasion:**
- Used a custom Mimikatz variant with a .NET loader to evade detection.
- Uploaded Dumpert tool to evade antivirus and EDR solutions.

**Credential Access:**
- Used Mimikatz to dump credentials from memory.
- Used Impacket tools for pass the hash technique.

**Lateral Movement:**
- Moved laterally using webshells and dumped credentials.
- Used Impacket's atexec to run commands on other systems.

**Command and Control:**
- Used AntSword webshell to communicate with the compromised server and run commands.
- Used AntSword Shell Manager to interact with the webshell.
