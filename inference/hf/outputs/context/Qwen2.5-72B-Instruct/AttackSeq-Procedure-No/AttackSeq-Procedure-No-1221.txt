     - Secondary shellcode stub decrypted and executed an embedded payload.
     - Payload created a temporary file in the %TEMP% folder and executed it using WinExec.
   - **Execution:**
     - Used igfxtray.exe to sideload a malicious DLL, hccutils.dll.
     - hccutils.dll performed security product checks and initiated the next stage of execution.
     - Used three techniques for persistence.
   - **Persistence:**
     - Modified registry keys to ensure malicious DLLs run after reboot or load with every user mode process.
     - Injected ResN32.dll into explorer.exe for persistent execution.
   - **Defense Evasion:**
     - Customized installation to avoid 24 identified security products.
     - Used multi-stage installation with checks to detect analysis by security researchers.
     - Created mutexes and named events to ensure only one instance was running.
   - **Collection:**
     - Captured data about the infected system.
     - Stole files of specific types from removable media.
     - Targeted Skype users, capturing encrypted data, screenshots, and Skype communications.
   - **Exfiltration:**
     - Compressed, encrypted, and stored data in specific directories or sent to the C2 server.
     - Logged file and directory operations, clipboard data, and encrypted network traffic.
