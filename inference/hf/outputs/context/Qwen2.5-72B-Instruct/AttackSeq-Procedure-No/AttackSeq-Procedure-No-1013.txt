### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- The campaign uses malicious delivery documents with social engineering to trick victims into enabling macros.
- Payloads are either embedded in the document or downloaded from the internet after the lure is clicked.
- A separate lure with a Java key generator is bundled with a Java downloader.

**Persistence:**
- The path to the VBS script is added to the task scheduler.

**Defense Evasion:**
- The campaign uses obfuscation techniques.
- PowerShell scripts and other files use publicly available code to bypass AppLocker.

**Discovery:**
- The backdoor collects machine information and takes screenshots.
- This data is sent to the C&C server.

**Collection:**
- The backdoor collects machine information and takes screenshots.
- This data is sent to the C&C server.

**Command and Control:**
- Communication with the C&C server is done via XML messages.
- The backdoor queries internet services to get the IP address and location of the infected machine.
- The communication is encrypted using a simple RSA algorithm.

**Others:**
- The attackers monitor incoming connections to the C&C server.
- False flags and error messages in simplified Mandarin Chinese are embedded in the PowerShell script.
