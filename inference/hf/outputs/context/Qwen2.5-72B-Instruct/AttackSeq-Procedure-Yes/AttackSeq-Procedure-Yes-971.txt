### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Weaponized RTF file exploiting RoyalRoad tool to drop 'intel.wll' loader.
- Archive files with a legitimate executable and a malicious DLL for DLL hijacking.
- Direct delivery via an executable file serving as a loader.

**Execution:**
- Aria-body loader establishes persistence, injects itself into processes, decrypts configuration data, contacts C&C servers, and loads the Aria-body backdoor.

**Persistence:**
- Established through the 'Startup' folder or 'Run' registry key.
- Some variants injected into processes like 'rundll32.exe' and 'dllhost.exe'.

**Defense Evasion:**
- C&C servers operated within limited daily windows.
- Fileless loading techniques.
- Frequent changes in server infrastructure and loader variants.

**Collection:**
- Capabilities include file management, screenshots, file search, process enumeration, file metadata collection, TCP session closure, OS information gathering, and location verification.

**Command and Control:**
- Used HTTP and TCP protocols.
- Sent data to C&C domain with XORed password.
- Listened for additional commands from the server.
