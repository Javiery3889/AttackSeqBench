### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Blind Eagle uses a JavaScript downloader with ActiveXObject to execute PowerShell commands.
- The script is hosted on Discord CDNs and downloaded from a specific URL.

**Execution:**
- A PowerShell script loads a DLL from an obfuscated and Base64 encoded string.
- The DLL contains an AMSI bypass method.
- The script checks for and creates a registry key if it doesn't exist.
- It drops two files: myScript.ps1 and SystemLogin.bat.
- A VBScript (Login1.vbs) is placed in the Startup folder to execute SystemLogin.bat on startup.
- An obfuscated batch script deobfuscates and executes myScript.ps1 using mshta.
- Another PowerShell script loads the AMSI bypass DLL and an AES256 Decryptor.
- The decrypted content is a new PowerShell script leading to njRAT (Bladabindi).

**Persistence:**
- A VBScript (Login1.vbs) in the Startup folder ensures the execution of SystemLogin.bat on system startup.

**Defense Evasion:**
- The PowerShell script includes an AMSI bypass method.
- Scripts and registry values are obfuscated to evade detection.

**Others:**
- APT Blind Eagle targets Colombian government institutions and corporations.
- They use sophisticated techniques and custom malware.
