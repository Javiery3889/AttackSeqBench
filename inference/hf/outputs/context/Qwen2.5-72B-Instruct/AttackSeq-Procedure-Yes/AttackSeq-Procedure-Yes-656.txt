### Step-by-Step Analysis

#### 1. Analyze the CTI Report

**Initial Access:**
- Gallmaker group initiated the attack by delivering malicious Office documents via spear-phishing emails.
- These documents exploited the Microsoft Office Dynamic Data Exchange (DDE) protocol to execute commands in memory, avoiding leaving artifacts on disk.

**Defense Evasion:**
- Gallmaker used living off the land (LotL) tactics and publicly available hack tools.
- They deleted tools from victim machines after use to avoid detection.
- Exploited the DDE protocol to execute code remotely without leaving artifacts on disk.

**Command and Control:**
- Used three primary IP addresses for C&C infrastructure.
- Likely used the WinZip console to archive data for exfiltration.

**Exfiltration:**
- Used the legitimate version of the WinZip console to archive data for exfiltration.

**Others:**
- Targeted government, military, and defense sectors.
- Active since December 2017, with recent activity in June 2018.
- Indicative of a cyber espionage campaign, likely state-sponsored.
- Symantec's TAA technology detected their activities.
- Protections include System Infected: Meterpreter Reverse TCP and W97M.Downloader.
- IoCs and Yara rules are shared for detection.
