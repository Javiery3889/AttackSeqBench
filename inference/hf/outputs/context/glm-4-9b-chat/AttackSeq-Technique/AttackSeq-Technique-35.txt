
Step 1: Analyze the CTI Outline
- Initial Access: Email attachment used as initial infection vector, social engineering to open .src file, display decoy document, execute malware.
- Persistence: Malware drops files and creates shortcuts for persistence.
- Defense Evasion: Malware evolves, checks for previous versions, and uses advanced decoy documents.
- Credential Access: Steals keystrokes and credentials from browsers.
- Collection: Steals files, keystrokes, and performs screenshots.
- Command and Control: Communicates with C2 using HTTP connections.
- Exfiltration: Uploads stolen data to C2 infrastructure.
- Others: Malware named KONNI, authored by Paul Rascagneres, targeting public organizations, using free web hosting for C2.

The attack sequence in the order presented by the MITRE ATT&CK tactics is:
1. Initial Access
2. Persistence
3. Defense Evasion
4. Credential Access
5. Collection
6. Command and Control
7. Exfiltration
