
Step 1: Analyze the CTI Outline
- Reconnaissance: The attack began with the delivery of malicious documents containing job offers, which activated malicious macros on the victims' home computers. The attackers used system utilities and unknown PE files for reconnaissance.
- Initial Access: The initial access was achieved through the opening of malicious documents sent via LinkedIn, Telegram, WhatsApp, and corporate email.
- Execution: The Lazarus Group executed malicious code using system utilities, a backdoor named CommsCacher, and the Windows command line.
- Persistence: The group created services and placed shortcuts in the startup folder for persistence.
- Privilege Escalation: Privileges were escalated by creating local administrator accounts and manipulating system processes and services.
- Defense Evasion: The group used various techniques to evade detection, including encryption and hiding malware.
- Discovery: The Lazarus Group collected information about users, network settings, and processes using system utilities like net user, adfind, tasklist.exe, and systeminfo.exe, and SMBMap utility.
- Command and Control: The malware used the standard HTTP protocol to connect to C2 servers.
- Others: The Lazarus Group is a government-sponsored APT group known for targeted phishing and has been operating since 2009.
