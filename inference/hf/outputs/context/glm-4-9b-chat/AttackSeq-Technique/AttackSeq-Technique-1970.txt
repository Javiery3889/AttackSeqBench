
Step 1: Analyze the CTI Outline
- Initial Access: The StellarParticle campaign gained access via SSH, used port forwarding for RDP, and VPNs, and exported Chrome browser passwords.
- Persistence: Used credential hopping, Delegated Administrator rights, WMI persistence, crontab entry, and O365 Service Principals.
- Defense Evasion: Stole browser cookies, renamed utilities, and systems to bypass MFA and masquerade as legitimate binaries.
- Credential Access: Stole web session cookies, used privileged and non-privileged accounts for RDP, and accessed data from Information Repositories.
- Discovery: Used AdFind, PowerShell cmdlets, and custom tooling to gather information from Active Directory and probed external services.
- Collection: Accessed data from Information Repositories.

The attack sequence in the order presented by the MITRE ATT&CK tactics is:
1. Initial Access
2. Persistence
3. Defense Evasion
4. Credential Access
5. Discovery
6. Collection
