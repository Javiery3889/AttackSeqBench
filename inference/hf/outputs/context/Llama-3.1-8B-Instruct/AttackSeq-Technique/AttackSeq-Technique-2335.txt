**Step 1: Analyze the CTI Outline**

The CTI outline describes the attack sequence of Operation Overtrap, a cyber attack targeting Japanese online banking users. The attack sequence is as follows:

1. Initial Access: 
   - Phishing links in spam emails leading to fake banking websites
   - Executable malware downloads from phishing pages
   - Malvertising that delivers malware via a custom exploit kit named Bottle Exploit Kit

2. Execution: 
   - The Bottle Exploit Kit exploits vulnerabilities CVE-2018-15982 and CVE-2018-8174 to execute code on victims' systems

3. Persistence: 
   - Cinobi establishes persistence by copying itself to the '%PUBLIC%' folder and installing as a Winsock Layered Service Provider
   - Modifies the spooler service configuration to 'SERVICE_AUTO_START'
   - Disables several services

4. Privilege Escalation: 
   - Cinobi uses a UAC bypass method via the CMSTPLUA COM interface during its third stage of infection to escalate privileges

5. Defense Evasion: 
   - Downloads legitimate applications and renames them
   - Removes files from the '\AppData\LocalLow\' directory and the original dropper file to evade detection

6. Credential Access: 
   - Steals banking credentials by compromising victims' web browsers for form-grabbing and modifying web traffic to and from targeted Japanese banking websites

7. Others: 
   - The malware communicates over the Tor proxy for downloading additional malware stages and configuration files
