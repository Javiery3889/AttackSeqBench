**Step 1: Analyze the CTI Outline**

The CTI outline describes a cyber attack ordered by MITRE ATT&CK tactics. The attack sequence is as follows:

1. Initial Access: Spear-phishing emails with malicious DOCX documents were used to gain initial access.
2. Execution: The attackers leveraged old Microsoft Office vulnerabilities and a chain of in-memory loaders to install a previously unknown backdoor on victims' machines.
3. Persistence: The attackers established persistence by creating a scheduled task to run the exported function 'StartW' from a DLL named '5.t' with 'rundll32.exe' once a day.
4. Discovery: The loader gathered detailed information about the victim's computer, including hostname, OS name and version, system type, user name, and MAC addresses of networking adapters.
5. Collection: The backdoor installed by the attackers had capabilities to collect a wide range of data from the victim's system.
6. Command and Control: The backdoor communicated with a C&C server using a configuration that contained the server IP and port.
7. Exfiltration: The backdoor was capable of exfiltrating data over the C&C channel.
